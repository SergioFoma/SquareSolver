# Square Solver
## *Эта программа занимается решением квадратного уравнения или его частных случаев.*

## **Загрузка программы на компьютер Через веб-интерфейс**
1. Чтобы скачать весь проект, нужно:

2. Перейти на главную страницу нужного репозитория на GitHub.

3. Нажать зеленую кнопку Code в правом верхнем углу списка файлов.

4. В открывшемся меню выбрать Download ZIP — начнётся скачивание проекта в виде ZIP-архива. Останется только распаковать архив на компьютере.

## **Запуск программы**
Чтобы скомпилировать файлы, в консоли необходимо ввести команду `make`

Чтобы запустить решение самого уравнения необхoдимо ввести в консоли `Build/result.exe`

Чтобы запустить проверку метода решения уравнения необходимо ввести в консоли `Build/result.exe --Test`

## **Ввод данных**
При успешной компиляции программы и ее запуске в консоли вывидится строка:
```
Введите коэффициенты квадратного уравнения( a, b и c соответсвенно) на следующей строке через пробел:
```
Вам нужно будет ввести коэффициенты уравнения через пробел.

## **Результат**
Если Вы вызывали именно решение квадратного уравнения, то в результате Вы получите строку:
```
a: (введный коэффициент) b: (введенный коэффициент) c: (введенный коэффициент)
```
Строка выше будет содержать коэффициенты a, b и c, которые Вы ввели.

Далее последует ответ, то есть решение уравнения, или сообщение о неправильном вводе данных. Например:
```c
Введите коэффициенты квадратного уравнения( a, b и c соответсвенно) на следующей строке через пробел: 

a: 1 b: -2 c: 1

Имеет единственный корень: 1
```

Если Вы вызывали тестирование функци решения уравнения, то в консоли будет выводиться статус об успешном решении уравнения с заданными коэффициентами, или сообщение о неправильности решения. Например:
```c
Complited successfully with coefficients 1.2 -123.7 11.1
Complited successfully with coefficients 0 0 0
Complited successfully with coefficients 0 1 0
Complited successfully with coefficients 0 0 1
```
