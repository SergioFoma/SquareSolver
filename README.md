# Square Solver
## *Эта программа занимается решением квадратного уравнения или его частных случаев.*

## **Загрузка программы на компьютер через терминал**

### **Чтобы скачать весь проект, нужно:**
1. В Вашем терминале необходимо указать директорию, куда вы хотите скачать данный проект.

2. Затем в терминале введите
```
git clone https://github.com/SergioFoma/SquareSolver.git
```

3. Дождитесь скачивания.

4. После чего будет создана рабочая директория аналогичная названию репозитория.

## **Запуск программы**
Чтобы скомпилировать файлы, в терминале необходимо ввести команду `make`

Чтобы запустить решение самого уравнения необхoдимо ввести в терминале `Build/result.exe`

Чтобы запустить проверку метода решения уравнения необходимо ввести в терминале `Build/result.exe --Test`

## **Ввод данных**
При успешной компиляции программы и ее запуске в терминале вывидится строка:
```
Введите коэффициенты квадратного уравнения( a, b и c соответсвенно) на следующей строке через пробел:
```
Вам нужно будет ввести коэффициенты уравнения через пробел.

## **Результат**
Если Вы вызывали именно решение квадратного уравнения, то в результате Вы получите строку:
```
a: (введный коэффициент) b: (введенный коэффициент) c: (введенный коэффициент)
```
Строка выше будет содержать коэффициенты a, b и c, которые Вы ввели.

Далее последует ответ, то есть решение уравнения, или сообщение о неправильном вводе данных. Например:
```c
Введите коэффициенты квадратного уравнения( a, b и c соответсвенно) на следующей строке через пробел: 

a: 1 b: -2 c: 1

Имеет единственный корень: 1
```

Если Вы вызывали тестирование функци решения уравнения, то в терминале будет выводиться статус об успешном решении уравнения с заданными коэффициентами, или сообщение о неправильности решения. Например:
```c
Complited successfully with coefficients 1.2 -123.7 11.1
Complited successfully with coefficients 0 0 0
Complited successfully with coefficients 0 1 0
Complited successfully with coefficients 0 0 1
```
